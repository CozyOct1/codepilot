Capabilities

围绕代码仓库 Agent 的完整工程链路

CodePilot 不是单一聊天框，而是把仓库上下文、任务执行、安全边界和可观测数据组织成可审计的开发系统。

01

仓库索引与语义检索

解析 Python、Markdown、JSON、YAML、前端源码等文本文件，写入 Chroma，并为 Agent 任务召回 Top-K 代码上下文。

02

LangGraph 任务编排

将 retrieve、plan、execute 拆成结构化节点，显式展示上下文检索、记忆注入、计划生成和工具调用。

03

ReAct 执行轨迹

记录 Thought / Action / Observation / Final，方便复盘 Agent 为什么这么做、调用了什么工具、结果如何。

04

安全工具调用层

文件访问限定在目标仓库内，Shell 命令基于允许列表执行，并拦截 sudo、rm、curl、wget 等高风险 token。

05

任务持久化与记忆

使用 SQLite + SQLModel 存储会话、任务、消息和工具调用；结合短期记忆和长期记忆提升多轮连续性。

06

MCP 与可观测性

通过 MCP Server 暴露工具能力，并用 Prometheus 记录任务数量、工具调用耗时和接口性能指标。

Safety Design

          
公网 Demo 与本地完整模式明确隔离

          
Agent 项目最容易被质疑的是安全边界。CodePilot 在线页直接展示 Demo Mode 的限制，说明哪些能力只在本地授权环境开放。

        

        

          

            
Demo Mode

            
    
              
  • 只读示例仓库，不接受任意本地路径
    • 
              
  • 展示工具调用日志，但不执行真实危险 Shell
    • 
              
  • 文件修改以 Patch Preview 展示，需要人工审批
    • 
              
  • LLM API Key 只存在服务端，本页面不暴露密钥
    • 
            

          

          

            
Local Mode

            
    
              
  • 用户在本机 clone 后运行完整 Agent 能力
    • 
              
  • 工具层限制仓库根目录，阻断越权路径访问
    • 
              
  • Shell 命令基于允许列表，并拦截高风险 token
    • 
              
  • 任务、工具调用和结果写入 SQLite 便于复盘
    • 
            

          

          

            
Production Direction

            
    
              
  • 仓库 workspace 隔离，按 repo_id 访问
    • 
              
  • Redis 队列异步执行索引和 Agent 任务
    • 
              
  • 任务超时、步数限制、IP 限流和审计日志
    • 
              
  • 容器沙箱执行测试或构建任务
    •
